# 智学数理学习平台

这是一个基于前后端分离架构的在线学习平台，实现了用户登录、帖子发布与互动、评论管理等功能。

## 技术栈
- **后端**: Spring Boot, Java
- **前端**: Vue.js, Vite
- **部署**: Nginx

## 项目结构
- `/src/main`: Spring Boot 后端代码
- `/zhixue-vue`: Vue.js 前端项目

## 部署方式
1.  后端打包为 Jar 包，运行在 8080 端口
2.  前端项目执行 `npm run build`，将 `dist` 目录部署到 Nginx
3.  Nginx 配置反向代理，将 `/api/` 请求转发到后端服务

## 项目功能
- 用户登录/注册
- 帖子发布与互动
- 评论管理
- 用户中心
