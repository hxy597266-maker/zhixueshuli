线性代数简介

线性代数是理工科、计算机、大数据、人工智能专业核心基础课程，主要研究线性空间、向量、矩阵、线性变换等线性结构问题。

区别于微积分研究连续变化，线性代数侧重离散、多维、多变量的关系建模，是工程计算与算法设计的重要数学工具。

核心学习内容

  • 行列式：定义、性质、展开、计算
  • 矩阵理论：矩阵运算、逆矩阵、秩、分块矩阵
  • 线性方程组：高斯消元、解的判定与结构
  • 向量组：线性相关、线性无关、极大线性无关组
  • 相似矩阵、特征值与特征向量、二次型

实际应用

  • 图像处理、矩阵变换、图像压缩
  • 机器学习、深度学习、数据降维
  • 工程力学、电路计算、运筹规划
  • 计算机图形学、三维坐标变换

行列式

行列式是方阵对应的一个数值，用于判定矩阵可逆、方程组解的存在性、向量组相关性等。

基本性质

  • 行列互换，行列式值不变
  • 两行（列）互换，行列式变号
  • 两行（列）成比例，行列式为 0
  • 某一行含公因子，可以提取到行列式外侧
  • 可逆矩阵 ⇔ 行列式不为 0

二阶、三阶行列式公式

二阶行列式： | a b | | c d | = ad - bc 三阶行列式对角线法则、展开式计算

矩阵与线性方程组

基础矩阵运算

  • 矩阵加减：同型矩阵对应元素相加减
  • 矩阵乘法：前行乘后列，对应相乘再求和
  • 矩阵转置：行变列、列变行
  • 逆矩阵：满足 \(AA^{-1}=A^{-1}A=E\)

秩与方程组

利用初等行变换将矩阵化为行阶梯形，求解线性方程组：

  • 系数矩阵秩 ≠ 增广矩阵秩：无解
  • 秩相等且等于未知数个数：唯一解
  • 秩相等且小于未知数个数：无穷多解
初等行变换： 1. 两行互换 2. 某一行乘以非零常数 3. 某一行倍数加到另一行
