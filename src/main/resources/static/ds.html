数据结构简介

数据结构是计算机科学与技术专业的核心基础课程，主要研究数据的**逻辑结构、存储结构以及基本操作算法**。

C 语言是实现数据结构的经典语言，能直接操作内存、指针、结构体，更贴近底层原理。

三大逻辑结构

  • 线性结构：线性表、栈、队列、串、数组
  • 树形结构：二叉树、二叉搜索树、堆、哈夫曼树
  • 图形结构：无向图、有向图、网

两大存储结构

  • 顺序存储：数组实现，连续内存空间
  • 链式存储：指针实现，离散内存空间

学习意义

提升程序设计效率、优化算法性能，是面试、考研、工程开发必备基础。

线性表

线性表是最基础的数据结构，数据元素之间存在一对一的线性关系。

顺序表（数组实现）

  • 优点：随机访问、查找快
  • 缺点：插入删除需要移动大量元素
  • 适用：数据固定、频繁查询的场景

单链表（指针实现）

  • 优点：插入删除效率高，无需移动元素
  • 缺点：不能随机访问
  • 适用：频繁增删、长度不固定的场景

C 语言链表节点定义

typedef struct Node { // 数据域 int data; // 指针域 struct Node *next; } Node, *LinkList;

栈与队列

栈 Stack

  • 特点：先进后出 FILO
  • 操作：入栈 push、出栈 pop、取栈顶
  • 应用：函数调用、括号匹配、表达式求值

队列 Queue

  • 特点：先进先出 FIFO
  • 操作：入队 enQueue、出队 deQueue
  • 应用：任务排队、消息队列、缓冲区
