常微分方程

课程简介 一阶微分方程 二阶微分方程

常微分方程简介

常微分方程（Ordinary Differential Equations，ODE）是微积分的重要应用分支，研究包含**一元函数及其导数**的方程。

它广泛用于物理、力学、电子、控制、生物、经济等领域，用于描述随时间/空间变化的动态系统。

核心内容

  • 一阶常微分方程：可分离、齐次、线性、伯努利方程
  • 二阶线性微分方程：齐次、非齐次、常系数解法
  • 微分方程解的存在唯一性定理
  • 线性微分方程组、幂级数解法

应用领域

  • 物理：运动、振动、电路、热传导
  • 工程：自动控制、信号处理、系统建模
  • 计算机：数值模拟、动力学系统
  • 生物/经济：种群增长、经济增长模型

一阶常微分方程

一阶微分方程只包含**一阶导数**，是最基础、最常用的类型。

主要类型与解法

  • 可分离变量方程：dy/dx = f(x)g(y)，分离变量后积分
  • 齐次方程：可化为 dy/dx = φ(y/x)，令 u=y/x 换元
  • 一阶线性方程：y’+P(x)y=Q(x)，用常数变易法/公式法
  • 伯努利方程：y’+P(x)y=Q(x)yⁿ，换元化为线性方程

一阶线性方程通解公式

y = e^(-∫Pdx) [ ∫Q e^(∫Pdx) dx + C ]

二阶常系数线性微分方程

二阶常系数线性微分方程是工程与物理中最常用的方程，结构清晰、解法固定。

标准形式

  • 齐次：y'' + py' + qy = 0
  • 非齐次：y'' + py' + qy = f(x)

解法步骤

  • 写出特征方程：r² + pr + q = 0
  • 根据根的类型（两不同实根、重根、共轭复根）写通解
  • 非齐次方程：齐次通解 + 特解

典型示例

方程：y'' - 3y' + 2y = 0 特征方程：r²-3r+2=0 → r1=1, r2=2 通解：y = C1 e^x + C2 e^(2x)
